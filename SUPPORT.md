# Support

## How to Get Help

### Bug Reports

If you found a bug, please [open an issue](https://github.com/HanielCota/grabix/issues/new?template=bug_report.yml) using the Bug Report template. Include as much detail as possible.

### Feature Requests

Have an idea? [Open a feature request](https://github.com/HanielCota/grabix/issues/new?template=feature_request.yml) using the Feature Request template.

### Questions and Discussions

For general questions, ideas, or conversations that aren't bugs or feature requests, use [GitHub Discussions](https://github.com/HanielCota/grabix/discussions).

Good topics for Discussions:

- How-to questions
- Architecture or design discussions
- Sharing what you've built with Grabix
- Proposing large changes before opening a PR

### Security Issues

**Do not open a public issue for security vulnerabilities.** See [SECURITY.md](SECURITY.md) for instructions.

## Before Asking

1. Check existing [issues](https://github.com/HanielCota/grabix/issues) and [discussions](https://github.com/HanielCota/grabix/discussions).
2. Read the [README](README.md) and [CONTRIBUTING](CONTRIBUTING.md) guides.
3. Try the latest version to see if your issue has been fixed.
